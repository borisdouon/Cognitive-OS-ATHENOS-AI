# Metrics Directory
## Phase: A | Source: Athenos_AI_Strategy.md#L105

This directory stores metrics for alpha cohort tracking.

## Core Metrics (Strategic_Reinforcements_Gap_Closures.md#L25)
- Cognitive Clarity Index
- Emotional Resilience Score  
- Habit Evolution Rate
- Focus Stability Percentage
- Time Saved (minutes/day)

## Structure
```
metrics/
├── alpha_cohort/
│   ├── user_001/
│   │   ├── daily_reports/
│   │   ├── observations/
│   │   └── outcomes/
│   └── ...
└── aggregated/
    └── weekly_summary.jsonl
```

## Collection Guidelines
- All metrics stored locally (on-device)
- No cloud sync without explicit consent
- One-click deletion supported
- Encrypted at rest
